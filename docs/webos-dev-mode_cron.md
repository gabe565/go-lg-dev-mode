## webos-dev-mode cron

Extends dev mode session on a given interval

```
webos-dev-mode cron [flags]
```

### Options

```
  -h, --help                help for cron
      --interval duration   Extend cron interval (default 24h0m0s)
```

### Options inherited from parent commands
